# Announcement Bar

A dismissible sticky announcement / notification bar for the top of your site —
sales, notices, links — with no theme editing. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

Renders a configurable bar at the top of every front-end page: message, optional
CTA button, custom background/text colours, and a dismiss button remembered for
7 days (localStorage). Sticky-on-scroll is a Pro toggle.

## Settings

**Settings → Announcement Bar** — enable, message, button text/URL, colours,
dismissible, sticky (Pro).
